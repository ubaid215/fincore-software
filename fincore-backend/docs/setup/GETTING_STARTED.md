# Getting Started with FinCore

## Prerequisites

| Tool | Version | Command to verify |
|------|---------|-------------------|
| Node.js | 22.x LTS | `node -v` |
| npm | 10+ | `npm -v` |
| Docker | 24+ | `docker --version` |
| Docker Compose | 2+ | `docker compose version` |
| Git | 2.40+ | `git --version` |
